# Changelog

本项目使用语义化版本记录主要发布节点。

## v0.8.0 - 2026-06-26

- 新增淘汰赛左右半区连线图，中间展示最终晋级/冠军席位。
- 自动按 32强、16强、1/4决赛、半决赛到决赛汇聚展示晋级路线。
- 三四名决赛独立展示，避免干扰冠军路线。
- 保留原阶段卡片列表，方便查看每场比赛时间、状态和比分。

## v0.7.0 - 2026-06-26

- 新增淘汰赛对阵与晋级路线模块。
- 自动从赛程接口渲染 32强、16强、1/4决赛、半决赛、三四名和决赛。
- 支持未确定球队席位显示“待定”，球队确定后自动显示队名、队徽和比分。
- 高亮下一场淘汰赛，并显示各阶段完成进度。

## v0.6.1 - 2026-06-19

- 修复 The Odds API 时间参数格式，移除 ISO 毫秒部分以符合 `YYYY-MM-DDTHH:MM:SSZ` 要求。
- 验证 `soccer_fifa_world_cup` 与 `soccer_fifa_world_cup_winner` 当前可用。

## v0.6.0 - 2026-06-19

- 新增第三方赔率 API 框架，预设对接 The Odds API。
- 新增 `/api/odds-sports`，用于检查赔率服务覆盖的足球赛事 key。
- 新增 `/api/odds?matchId=...`，用于按当前比赛详情懒加载胜平负赔率。
- 比赛详情新增“赛前赔率”模块；未配置赔率 API Key 或暂无匹配数据时显示友好状态。
- 增加赔率免责声明：赔率仅供赛事信息参考，不构成投注建议。

## v0.5.0 - 2026-06-19

- 新增比赛卡片详情展开，显示完整开赛时间、阶段、小组、半场比分、赛果、裁判和数据更新时间。
- 新增赛程与积分榜的浏览器本地缓存兜底，接口失败时保留上次成功数据。
- 新增 SEO 与社交分享基础元信息。
- 为后续赔率模块预留详情展示位置。

## v0.4.0 - 2026-06-19

- 新增今日比赛、下一场倒计时、球队筛选、收藏赛事、手动刷新和自动刷新。
- 新增服务端短时 CDN 缓存，降低第三方 API 配额消耗。

## v0.3.0 - 2026-06-19

- 将赛程升级为中文球队名、队徽、比分和状态卡片。
- 新增赛程状态筛选、小组积分榜标签切换和完整积分列。
- 优化桌面双列与手机单列布局。

## v0.2.0 - 2026-06-19

- 迁移到 Netlify。
- 增加 Netlify Functions：`/api/matches` 与 `/api/standings`。
- 配置 GitHub 自动部署。

## v0.1.0 - 2026-06-19

- 创建初始 `index.html`。
- 发布基础 2026 世界杯观赛中心页面。
